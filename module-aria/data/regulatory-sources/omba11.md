# OMB Circular A-11 — Preparation, Submission, and Execution of the Budget

**Source ID:** `omb-a11`
**Authority:** Office of Management and Budget
**Scope (CLEAR):** Budget formulation, exhibit structure, and justification completeness.

> SYNTHETIC SUMMARY — not a legally complete restatement of OMB Circular A-11.
> These are the specific, deterministic rules the CLEAR engine evaluates an output
> against. They are a working subset for demonstration on synthetic data only.

## Rules CLEAR evaluates

1. **R-A11-1 — Justification narrative present.** Every budget exhibit and program
   submission must carry a written justification narrative. An exhibit submitted
   without a justification narrative is non-compliant.

2. **R-A11-2 — Exhibit type declared.** Every output presented for export must
   declare its exhibit / document type so it can be matched to the correct A-11
   format requirements. An output with no declared type is non-compliant.

3. **R-A11-3 — Congressional submissions held to the data-quality threshold.**
   Materials destined for a congressional submission must meet the platform
   data-quality threshold (90%). A congressional submission below threshold is a
   priority (P1) compliance violation.

## Notes

A-11 compliance is a precondition for any budget exhibit leaving the platform via
the SCRIBE export gate. CLEAR records the evaluation; a named human reviewer
certifies the output before export.
