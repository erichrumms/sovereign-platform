Required-Before-Production Design Obligations
R1 — Human Review Volume Architecture
Required before Stage 5. NEXUS at program or enterprise-scale volume cannot sustain individual review of every AI output.
Design: risk-stratified tiers · statistical sampling methodology · automatic escalation thresholds · sovereign-shell review queue component
R2 — AI Provider Abstraction Layer
Required in Stage 1. No product calls Anthropic API directly. sovereign-api-client wraps all LLM calls.
Resolve: GovCloud LLM provider decision · FedRAMP authorization for CUI tier · three-tier fallback as platform standard
R3 — Agent Operator Role
Required before Stage 2. Role must be formally defined and assigned before Security Framework alerts go live.
Produce: Agent Operator scope document · formal role assignment · succession procedure · production capacity plan
R4 — Federal Client Workforce Transition
Required before any client pilot. Federal transition state is 2–4 years. Implementation methodology must address it.
Produce: Client Implementation Methodology document · FLOWPATH pilot framing · reviewer capacity design · middle management transition protocol
